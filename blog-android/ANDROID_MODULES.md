# 风月博客 Android 端 - 功能模块清单

## 一、已完成模块

| 模块 | 说明 | 对齐 Uniapp |
|------|------|-------------|
| **登录 / 注册** | 用户名+密码登录，注册支持昵称 | pages/auth/login、register |
| **首页** | 快捷入口：写文章、留言弹幕墙、群聊大厅 | pages/index/index |
| **文章列表** | 下拉刷新、分页扩展 | pages/article/list |
| **文章详情** | 标题、元信息、Markdown 正文、图片加载、评论列表 | pages/article/detail |
| **评论** | 发表评论、**显示用户昵称** | - |
| **点赞 / 收藏 / 打赏** | 文章详情页操作 | - |
| **发布文章** | 标题、摘要、内容、**选择已有标签 + 自定义新标签**、图片上传 | pages/article/edit |
| **我的文章** | 当前用户发布列表 | pages/article/my |
| **我的收藏** | 收藏文章列表 | pages/favorite/list |
| **编辑资料** | 昵称、邮箱、手机号 | pages/profile/edit |
| **头像上传** | 选择图片上传并更新资料 | - |
| **排行榜** | 风月币排行 | pages/rank/index |
| **留言弹幕墙** | 留言列表 + 发送 | pages/message/wall |
| **群聊大厅** | WebSocket 实时群聊 | pages/chat/room |
| **钱包 / 签到** | 余额、签到领币 | - |
| **左滑返回** | 移动端手势 | - |
| **加载动画** | 品牌色 ProgressBar、部分页面加载遮罩 | - |

## 三、技术备注

- **评论昵称**：后端 `/portal/comment/list` 与 `submit` 已返回 `nickname`，Android 优先展示昵称，无则兜底为「用户 ID」或「游客」。
- **标签**：发布时可选择已有标签，亦可输入新标签名称添加，提交时传 `tagIds` + `newTags`。
